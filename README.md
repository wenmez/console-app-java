**«Студенттердің үй тапсырмасын есепке алу» жүйесі**

Негізгі мақсат

Оқытушылар мен студенттер арасындағы үй тапсырмаларын басқару, тағайындау, орындауды белгілеу және бағалау процестерін жеңілдететін, қарапайым және тиімді консольдік жүйе құру.

Міндеттері
1. Деректерді басқару: Студенттер мен тапсырмалар туралы ақпаратты сақтауды қамтамасыз ету.
2. Рөлдік айырмашылық: Әкімші (оқытушы) мен Студенттің функцияларын нақты бөлу.
3. Есепке алу: Тапсырмаларды тағайындау, орындалғанын белгілеу және бағалау мүмкіндігі.
4. Архитектуралық талаптар: ОББ қағидаларына және Бір Жауапкершілік Принципіне (SRP) сәйкес келетін таза кодты әзірлеу.
